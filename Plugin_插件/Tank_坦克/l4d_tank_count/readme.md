# 中文說明
Tank死亡時顯示Tank存活多長時間、對倖存者造成的 倒地/死亡/總傷害/拳頭/石頭/車子 統計表

* 圖示
	* 統計表
	<br/>![l4d_tank_count_1_zho](image/zho/l4d_tank_count_1_zho.jpg)

* 原理
	* 適用於對抗模式，但只支援場上一隻Tank的統計
	* 適用於戰役/生存/寫實模式，支援場上多隻Tank的統計
