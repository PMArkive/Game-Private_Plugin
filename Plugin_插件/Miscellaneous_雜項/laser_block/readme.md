# 中文說明
遊戲開始後不能升級紅外線雷射

* 原理
	* 遊戲開始後地圖上的紅外線雷射升級裝置不能使用
		* 這裡指的"遊戲開始"為倖存者離開安全區域或是生存模式計時開始
