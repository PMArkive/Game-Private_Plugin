# 中文說明
只追第一個驚嚇她的人，不管多少人阻擋她的路，Witch永遠不會改變目標

* 原理
	* Witch只會追第一個驚嚇她的人
	* Witch在追逐的路上，不管多少隻特感或多少個倖存者阻擋她的路，Witch永遠不會改變目標
