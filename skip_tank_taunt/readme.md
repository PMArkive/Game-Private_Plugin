# 中文說明
Tank爬行障礙物速度變快 + 略過咆哮勝利動畫

* 原理
	* 戰役模式之下的Tank打中倖存者之時偶而會有咆哮姿勢，浪費Tank攻擊時間

* 功能
	1. 調整爬行速度
	2. 開關咆哮勝利動畫
